# Telegram Channel Reader

A lightweight tool for retrieving posts from Telegram channels for data analysis and content pipelines.

Supports:

- reading messages from channels
- detecting URLs
- retrieving metadata
- incremental updates

Designed to be used by AI pipelines and automation systems.

---

## Features

read messages from Telegram channels

detect new posts since last processed ID

extract URLs and metadata

support integration with LLM pipelines

---

## Use cases

content curation

knowledge ingestion pipelines

AI editorial assistants

news aggregation systems
